# ZeriEngine

## Requisiti di compilazione

### Windows
- Visual Studio 2022 17.5+ (MSVC con supporto C++23)
- CMake 3.28+
- vcpkg

### Linux
- GCC 14+ oppure Clang 17+
- CMake 3.28+
- vcpkg
- Nota: GCC 13 non supporta `std::print`. GCC 14 è richiesto.

### macOS
- Xcode 15+ (AppleClang con supporto C++23 completo)
- CMake 3.28+
- vcpkg
- Nota: macOS 13 Ventura o superiore richiesto per Xcode 15.
